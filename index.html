GitHub Repository Készítés

Az oldal lényege, hogy megtanuljuk, hogyan kell létrehozni egy GitHub könyvtárat.
  1. Az első lépés, hogy létrehozunk egy GitHub fiókot, ezt beállítjuk, testreszabjuk, ahogy szeretnénk.
  2. Rákattintunk a jobbfelső sarokba, majd "Your repostories"-ra, majd szintén a jobb felsősarokban a "New"-ra.
  3. Következő lépésben adunk egy nevét a "reponknak", leírást is hozzáadhatunk, a publicitását is kiválaszthatjuk, hogy mindenki lássa vagy csak saját magad, és hozzátudunk adni egy README fájlt is.
